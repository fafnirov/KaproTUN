# Screenshots для README

`README.md` и `README.ru.md` ссылаются на эти два файла:

- `main-window.png` — главное окно приложения, **рекомендуемая
  ширина 1200-1400 px**, тёмная тема, желательно с подключённым
  конфигом чтобы был виден живой UI (connect button в зелёном
  состоянии, traffic stats, ping). На гитхабе картинка ужмётся до
  ~640 px, но исходник большой даёт чёткость на ретине.

- `tray.png` — раскрытое меню системного трея (правый клик по иконке
  KaproVPN в трее). Должно быть видно: «Подключиться / Отключиться»
  вверху, топ-3 быстрых конфига, разделители, выход. **Рекомендуемая
  ширина ~600-700 px** (трей-меню маленькое — ширина = ширина меню).

## Как снять

**Windows:**
1. Открой KaproVPN, подключись к какому-нибудь конфигу.
2. Для главного окна: `Win+Shift+S` → выдели окно → сохрани через
   Paint / любой image editor как `main-window.png`.
3. Для трея: правый-клик по иконке в трее (рядом с часами) → пока
   меню открыто, нажми `PrtSc` → вставь в Paint → обрежь
   до меню → сохрани как `tray.png`.

Подложи эти два файла сюда (`docs/screenshots/`) и закоммить — README
сразу станут показывать твою картинку на гитхабе.

## Анонимизация

Перед коммитом **проверь что на скриншотах не видно**:

- Реальных IP-адресов твоего VPN-сервера (закрой / заблюрь поле
  «адрес сервера» в карточке конфига, или используй конфиг
  с обобщённым именем типа `Test-VLESS` для скриншота).
- Subscription URL с твоим private token'ом (если он виден в
  Settings → Subscription).
- Имени Windows-пользователя в Path-полях (если на скриншоте видны
  пути типа `C:\Users\твоё-имя\...`).

Скриншоты публикуются на github.com под GPL вместе с кодом — то, что
ты тут заблюришь, никто никогда не увидит. То, что не заблюришь —
проиндексирует Google.
